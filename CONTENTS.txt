GETTING STARTED:
================

1. Install Node.js:
   Download from https://nodejs.org/

2. Install Dependencies:
   cd 3d-viewer
   npm install

3. Start Development Server:
   npm run dev
   (Opens http://localhost:3000)

4. Build for Production:
   npm run build
   (Creates dist/ folder)
